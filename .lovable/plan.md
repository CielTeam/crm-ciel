

# What's Next — Phase 3 Recommendations

Here's a summary of what's been built and what remains:

## Completed
- Auth with Auth0 + Supabase sync
- Dashboard shell with role-based sidebar
- Tasks module (CRUD, assign, priority, status)
- Leaves module (request, review, balances)
- Messages module (conversations, threads)
- Calendar module (month/week/day views with tasks & leaves)
- Directory module (employee listing with filters)
- Notifications module (automated alerts, click-through)
- Admin Console (user & team management)
- Audit Logs
- Settings page (read-only profile)

## Recommended Next Phase — Polish & Productivity

| Priority | Feature | Description |
|----------|---------|-------------|
| 1 | **Settings page buildout** | Profile editing (display name, phone, avatar upload), notification preferences, theme toggle (dark/light) |
| 2 | **Dashboard activity feed** | Replace the placeholder with a real feed showing recent tasks, leaves, messages, and notifications |
| 3 | **Task filters & sorting** | Filter tasks by assignee, status, priority, due date; add sorting options |
| 4 | **Real-time notifications** | Use Supabase Realtime subscriptions for instant notification badge updates instead of polling |
| 5 | **Profile editing from Directory** | Allow admins to edit user roles/teams directly from the directory or admin console |
